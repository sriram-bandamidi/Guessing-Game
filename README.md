# Guessing Game

A simple interactive number guessing game built using HTML, CSS, and JavaScript.

## Features

* Random number generation
* User input validation
* Instant result checking
* Interactive UI design
* Responsive layout

## Tech Stack

* HTML5
* CSS3
* JavaScript
